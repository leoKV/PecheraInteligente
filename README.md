# Pechera Inteligente Para Perros
## Objetivo
Diseñar un dispositivo inteligente que permita la detección de la ubicación, actividad fisica,
temperatura corporal, humedad, sonido, proximidad y cantidad de luz a la que se expone el perro.
Posteriormente los datos seran deplegados en una pantalla inteligente para el monitoreo adecuado 
del cuidado del perro.
## Beneficiario
- Dueños de perros.

Esto sera util para detectar la actividad inusual del perro y determinar el nivel de humedad de mi perro para poder controlarla.|Debe hacerse|1|El dueño del perro le colocará la pechera inteligente al perro en cuestión, para posteriormente visualizar los datos de la humedad del perro, por medio de una pantalla inteligente incorporada en la pechera.|José Manuel Matehuala Avíles|
